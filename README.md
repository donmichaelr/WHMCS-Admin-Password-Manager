# WHMCS Admin Password Manager

* One Centralized Place to Securely store your passwords for admins to access.
* Option to specify which other admins have read/write access.
* options to link entry to a specific client. Good for storing social media login details or google adwords account logins for each client.
* Search function to search through password entries.
* Time Saving - get rid of the spreadsheets.
* Never lose a password again.
